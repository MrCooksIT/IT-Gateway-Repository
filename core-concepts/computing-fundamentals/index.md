# Computing Fundamentals

This section covers the foundational concepts of computing that form the basis for all IT studies.

## Overview

Computing fundamentals introduce you to the basic concepts and principles that underpin all computing systems. These topics appear early in the Grade 10 curriculum but remain relevant throughout your IT studies.

## Topics in this section

### [Computer Basics](./computer-basics)
- Definition of Information and Communication Technologies (ICTs)
- The general model of a computer system
- Types of computers and their classifications
- Advantages and disadvantages of using computers

### [Data Representation](./data-representation)
- Binary, decimal, and hexadecimal number systems
- Conversion between number systems
- Digital character representation (ASCII/UTF-8, Unicode)
- Primitive data types and storage

### [Number Systems](./number-systems)
- Detailed exploration of binary and hexadecimal
- Number system conversions
- Mathematical operations in different number systems
- Applications in computing

### [Information Processing](./information-processing)
- Distinction between data and information
- The information processing cycle
- Data collection, processing, and output
- How computers transform data into meaningful information

## Key Concepts

{% include figure.html src="computing-model.png" alt="Basic Computing Model" caption="The basic computing model showing input, processing, output, storage, and communication" %}

The fundamental computing model includes:
- **Input**: Data entry into the system
- **Processing**: Manipulation of data according to instructions
- **Storage**: Retention of data and programs
- **Output**: Presentation of processed information
- **Communication**: Exchange of data between systems

## Learning Objectives

After studying these topics, you should be able to:
1. Explain the basic structure and function of computer systems
2. Convert between different number systems
3. Understand how data is represented in computer systems
4. Distinguish between data and information
5. Describe the information processing cycle
